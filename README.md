# Coding Conventions
- Variables: camelCase
```
int myInteger;
string myStringForMyProgram;
```

- Functions: UpperCase  
```
void MyFunction() {}
int MySecondFunction() {}
```

- Classes: UpperCase

```
//Vector3D.h

    private:
        int x;
        int y;
        int z;
    public:
        Vector3D::Vector3D(int i, int j, int k) {
          ...
        }

        void PrintComponents() {
          ...
        }
}
```
